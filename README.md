# PruebaDiagnostica-TSI-511-ProgramacionIII  

Se resuelve el ejercicio de prueba diagnostica.  
Desarrolle un Programa que le permita al restaurante “Tradición Llanera” generar el valor de la cuenta a cobrar.  
La cuenta se calcula como: valor base de consumo + impuesto al consumo + propina voluntaria  
- El impuesto al consumo es del 8% del valor base.
- La propina voluntaria es del 5% del valor base, si el cliente acepta pagar.
